## English

Welcome to the feedback repository for [Ramadan Countdown](https://ramadancountdown.top/)!

### About

This repository is dedicated to collecting feedback, bug reports, and feature requests for the Ramadan Countdown website.

### How to Submit Feedback

If you encounter any issues while using the website or have suggestions for improvement, please follow these steps:

1. **Check existing issues**: Before creating a new issue, please search the [existing issues](https://github.com/Esperanza1115/ramadan-countdown-feedback/issues) to see if your problem has already been reported.

2. **Create a new issue**: If your issue hasn't been reported yet, click the [New Issue](https://github.com/Esperanza1115/ramadan-countdown-feedback/issues/new) button to create one.

3. **Provide details**: When creating an issue, please include:
   - A clear and descriptive title
   - Steps to reproduce the problem (if applicable)
   - What you expected to happen
   - What actually happened
   - Your browser and device information
   - Screenshots (if applicable)
